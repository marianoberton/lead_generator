Como registrarse en cada servicio

Pasos exactos para conseguir la API key gratuita en cada plataforma. Repeti el proceso con cada una de tus cuentas de email.

Skrapp.io

MEJOR OPCION

100 emails/mes gratis por cuenta = 1,200 con 12 cuentas

100/mes
  1. 1

    Crear cuenta en skrapp.io

    Ir a skrapp.io → Sign Up → registrarte con tu email

  2. 2

    Ir a Settings → API

    En el menu superior derecho → Settings → seccion API

  3. 3

    Copiar la API Key

    Copiar el token y pegarlo arriba seleccionando servicio "Skrapp"

Formato: Solo API Key (sin secret)

Snov.io

50 creditos/mes gratis por cuenta = 600 con 12 cuentas

50/mes
  1. 1

    Crear cuenta en snov.io

    Ir a snov.io → Register → registrarte con tu email

  2. 2

    Ir a Settings → API

    Profile → Integrations → API → copiar Client ID y Client Secret

  3. 3

    Cargar las dos credenciales

    En el form de arriba: API Key = Client ID, Secret = Client Secret

Formato: API Key = client_id + Secret = client_secret
